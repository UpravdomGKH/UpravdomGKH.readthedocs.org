Публикация организации на Реформу ЖКХ
-----------------------------------------

Для того чтобы опубликовать организацию на Реформу ЖКХ. Отметьте галочкой требуемую организацию.

.. image:: ../_images/02-employment-section-organization/13.png

Затем нажмите на кнопку «Отправить».

.. image:: ../_images/02-employment-section-organization/14.png

 Статус отправки вы можете посмотреть в разделе «Журнал событий»
