Для чего предназначен принтер ГИС ЖКХ?
=============
Принтер ГИС ЖКХ – это виртуальный принтер, который может «печатать» (публиковать) квитанции в систему Управдом ЖКХ для упрощения процесса передачи данных о лицевых счетах в федеральную систему ГИС ЖКХ.
Теперь нет необходимости в ручном вводе информации о каждом лицевом счёте, принтер ГИС ЖКХ внесёт её самостоятельно.
