Для чего предназначен принтер ГИС ЖКХ?
=============
Принтер ГИС ЖКХ – это виртуальный принтер, который может печатать документы в систему Управдом ЖКХ для дальнейшей работы с ними. Он облегчает процесс передачи данных о лицевых счетах в федеральную систему ГИС ЖКХ.
Теперь нет необходимости в ручном вводе информации о каждом лицевом счёте, принтер ЖКХ внесёт её сам.
