Для чего предназначен принтер ГИС ЖКХ?
=============

**Принтер ГИС ЖКХ** – это виртуальный принтер, который может «печатать» (публиковать) квитанции в сервис «Управдом ЖКХ» для упрощения процесса передачи данных о лицевых счетах в федеральную систему «ГИС ЖКХ».
Теперь нет необходимости в ручном вводе информации о каждом лицевом счёте в ГИС ЖКХ, достаточно опубликовать квитанции с помощью принтера в сервис «Управдом ЖКХ», после чего загрузить лицевые счета в ГИС ЖКХ.
