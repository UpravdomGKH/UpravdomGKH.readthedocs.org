Для чего предназначен принтер ГИС ЖКХ?
=============

**Принтер ГИС ЖКХ** – это виртуальный принтер, который выполняет «печать» (публикацию) квитанции в сервис «Управдом ЖКХ» для упрощения процесса создания платежного документа, создания (обновления) лицевого счета, создания (обновления) помещения, 
для последующей публикации данных в федеральную систему «ГИС ЖКХ».
За счет использования принтера «ГИС ЖКХ» нет необходимости ручного размещения информации по каждому лицевому счету, платежному документу. Достаточно выполнить печать квитанций, а после чего загрузить данные по лицевым счетам и опубликовать платежные документы в «ГИС ЖКХ ЖКХ».
