Как работать с показателями приборов учета в системе?
=============
Журнал «Объемы потребления и начисления ОПУ» предназначен для загрузки, хранения и обновления, при помощи загрузки файла excel, данных по объемам потребления и начислениям ОПУ.

Журнал «Объемы потребления ИПУ» предназначен для загрузки, хранения и обновления, при помощи загрузки файла excel, данных по объемам потребления ИПУ.

Журнал «Показания ОПУ» предназначен для загрузки показаний по общедомовым приборам учета в систему и последующей публикации данных приборов в ГИС ЖКХ.
